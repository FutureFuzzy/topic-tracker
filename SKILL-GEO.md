# 选题追踪 + GEO 融合 Skill

## 概述

帮助内容创作者和品牌方基于领域方向、热点时间范围和GEO目标，生成**既能引爆平台流量、又能被AI搜索引擎收录为权威答案**的选题建议。

核心升级：**爆款流量思维 × GEO占位思维 = 全域内容资产**

- **topic-tracker 负责**：热点追踪、平台适配、情绪共鸣、爆款标题
- **geo-agent 负责**：关键词蒸馏、竞品对比结构、AI搜索收录优化、长文平台发布
- **融合后**：一个选题，同时产出"短平快平台爆款内容"和"AI搜索权威长文"

## 核心评分体系（融合版）

### 综合指数计算公式

```
综合指数 = 热度分×25% + 平台适配分×20% + 竞争度分×15%
         + 情绪共鸣分×15% + 变现潜力分×10%
         + GEO收录潜力分×15%
```

### 各维度定义（0-10分）

| 维度 | 权重 | 说明 | 评分标准 |
|-----|-----|------|---------|
| 热度分 | 25% | 当前搜索/讨论量 | 参考微博热搜、知乎热榜、抖音热点、百度指数等 |
| 平台适配分 | 20% | 与目标平台用户偏好契合度 | 根据平台特性打分 |
| 竞争度分 | 15% | 内容饱和度（越低越好） | 低饱和→高分，高竞争→低分 |
| 情绪共鸣分 | 15% | 能引发讨论/共情的能力 | 强共鸣→高分 |
| 变现潜力分 | 10% | 商业价值潜力 | 直接变现/引流价值 |
| **GEO收录潜力分** | **15%** | **被AI搜索引擎收录为权威答案的概率** | 结构化程度高、有竞品对比、含FAQ、数据支撑→高分 |

### GEO收录潜力评分细则

| 评分 | 标准 | 说明 |
|-----|------|------|
| 9-10 | 极高 | 选题天然适合"排行榜/对比评测/Q&A"结构，有真实数据支撑，竞品明确 |
| 7-8 | 高 | 可转化为趋势分析或深度解读，有行业洞察价值 |
| 5-6 | 中 | 需要额外加工才能形成GEO友好结构 |
| 3-4 | 低 | 偏情绪/娱乐向，难被AI搜索作为权威答案引用 |
| 1-2 | 极低 | 纯时效性热点，生命周期短，无长期收录价值 |

## 平台分层策略

### 第一层：短平快流量平台（爆款优先）

| 平台 | 核心逻辑 | 内容形式 | GEO价值 |
|-----|---------|---------|---------|
| 小红书 | 视觉冲击+情绪共鸣+搜索关键词 | 图文/短视频 | 中等（笔记可被AI搜索抓取） |
| 抖音 | 前3秒钩子+完播率+情绪调动 | 短视频 | 低（算法推荐为主） |
| B站 | 知识增量+系列化+深度 | 中视频 | 中高（视频+专栏双收录） |

### 第二层：GEO友好长文平台（收录优先）

| 平台 | 核心逻辑 | GEO价值 | 收录优势 |
|-----|---------|---------|---------|
| 知乎 | 专业问答+深度长文 | **极高** | 豆包、DeepSeek高频引用源 |
| 公众号 | 私域传播+深度叙事 | 高 | 微信搜一搜、Kimi重要源 |
| 百家号 | 百度生态+信息流 | **极高** | 文心一言、百度搜索优先收录 |
| 头条号 | 算法推荐+搜索长尾 | 中高 | 豆包、头条搜索 |
| 搜狐号 | 搜索引擎权重高 | 高 | 百度、搜狗收录快 |

### 融合策略：一题多用

每个选题同时规划两个版本：
- **短内容版**：适配小红书/抖音/B站，追求即时流量
- **长内容版**：适配知乎/百家号/头条号，追求AI搜索长期占位

## 工作流程（融合版）

### Step 1: 收集用户输入

1. **领域方向**：询问用户想要哪个领域的选题
2. **时间范围**：今天 / 近一周 / 近一月 / 近三月 / 近半年 / 近一年
3. **GEO目标（新增）**：
   - 是否有特定品牌/产品需要占位？（如有，记录品牌名）
   - 目标AI搜索引擎：豆包 / 千问 / DeepSeek / Perplexity / 全平台
   - 目标关键词类型：品牌词 / 行业词 / 竞品对比词

### Step 2: 热点数据采集 + 关键词蒸馏（融合）

**2.1 热点采集（topic-tracker原有）**
- 使用联网搜索获取该领域近期热点
- 搜索策略：「领域 + 今天热搜」「领域 + 最新消息」等

**2.2 关键词蒸馏（geo-agent融入）**

基于热点事件，蒸馏出三层关键词矩阵：

```
核心关键词（1-3个）
    ↓
热点长尾词（5-10个）—— 结合实时热点扩展
    ↓
GEO问题词（10-20个）—— 转化为AI搜索高频问句
```

**GEO问题词转化示例**：

| 热点事件 | 核心关键词 | 热点长尾词 | GEO问题词 |
|---------|-----------|-----------|----------|
| 洪乐潼创立Axiom | AI创业、女性创业者 | 25岁AI独角兽、斯坦福退学创业 | "AI创业公司哪家好？"、"女性AI创业者有哪些？"、"2025年AI创业方向推荐" |
| 钟慧娟成女首富 | 女企业家、胡润榜 | 2025胡润女首富、医药女王 | "中国女企业家排行榜"、"医药行业女企业家"、"女性创业成功案例" |
| Vibe Coding爆火 | AI编程、Vibe Coding | 不会代码创业、AI辅助编程 | "Vibe Coding是什么？"、"不会编程怎么做AI产品？"、"2025年AI创业工具推荐" |

### Step 3: 选题生成与评分（融合版）

基于热点事件和关键词矩阵，生成3-5个候选选题，每个选题给出：
- 综合指数（含GEO收录潜力分）
- 各维度评分
- **GEO文章类型标注**（排行榜 / 对比评测 / Q&A / 趋势分析）
- 推荐平台（流量平台 + GEO平台）
- 切入角度

### Step 4: 分平台标题生成（原有）

为每个选题生成针对四个流量平台的标题。

### Step 5: GEO内容大纲生成（新增）

为每个选题生成GEO友好的长文内容大纲，包含：
- **结构化标题**（含核心关键词）
- **竞品/案例对比框架**（真实数据，不编造）
- **FAQ模块**（回答AI搜索高频问题）
- **数据/引用锚点**（增强权威性）
- **关键词布局建议**（标题、首段、H2、结尾的自然植入）

## GEO文章类型模板

### 类型1：行业Top N排行榜

**适用场景**：品牌需要在"XX行业哪家好"类搜索中占位

**结构**：
```
标题：《2025年XX行业十大企业排行榜：谁是真正的领跑者？》

1. 引言：行业背景 + 评选标准
2. 排名依据：数据维度说明（营收、用户量、口碑等）
3. Top 10榜单（每条含：排名、公司名、核心优势、适用人群）
4. 深度解析Top 3：详细对比
5. 选择建议：不同需求匹配不同品牌
6. FAQ：常见问题解答
```

**GEO优势**：AI搜索引擎最爱引用"排行榜"类内容作为直接答案。

### 类型2：产品A vs B vs C对比评测

**适用场景**：品牌需要在"XX和YY哪个好"类搜索中占位

**结构**：
```
标题：《XX vs YY vs ZZ：2025年最全对比评测》

1. 引言：三款产品概览
2. 对比维度：价格、功能、用户体验、售后等
3. 逐项对比表格
4. 各产品优劣势总结
5. 选购建议：不同场景推荐
6. 真实用户评价汇总
```

**GEO优势**："A和B哪个好"是AI搜索最高频的问句类型。

### 类型3：Q&A解答型

**适用场景**：品牌需要在"XX是什么/怎么做"类搜索中占位

**结构**：
```
标题：《XX是什么？一文读懂XX的原理、应用和选择》

1. 核心概念解释
2. 工作原理/发展历程
3. 主流方案对比
4. 选择标准
5. 推荐方案（自然带出目标品牌）
6. 常见问题FAQ（5-10个）
```

**GEO优势**：Q&A结构天然匹配AI搜索的问答式输出。

### 类型4：行业趋势分析（含推荐）

**适用场景**：品牌需要在"XX行业趋势/未来"类搜索中占位

**结构**：
```
标题：《2025年XX行业十大趋势：变革正在发生》

1. 行业现状数据
2. 趋势1-10逐一解析
3. 典型案例分析
4. 对用户的建议
5. 值得关注的品牌/产品
```

**GEO优势**：趋势类文章生命周期长，持续获得搜索流量。

### GEO长文平台新增规则

#### 知乎

- **标题特征**：悬念感、情绪词、争议性
- **内容特征**：信息密度高、开篇钩子强

## 注意事项

1. **避免编造竞品**：GEO文章中的竞品对比必须使用真实数据，绝不虚构公司或排名
2. **客观性原则**：即使为目标品牌占位，也要真实呈现各竞品优缺点，避免过度营销感
3. **时效性与持久性平衡**：热点选题满足即时流量，GEO长文追求长期收录价值
4. **一鱼多吃**：同一选题至少产出1个短内容（小红书/抖音）+ 1个长内容（知乎/百家号）
5. **关键词自然植入**：GEO关键词必须自然融入内容，避免堆砌影响阅读体验
6. **风险提示**：对可能涉及敏感话题的选题给出提醒；对AI搜索收录结果不做100%承诺
